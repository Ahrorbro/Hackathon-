# Design — Wait App

Place design assets here when they’re sent (e.g. **PDF** or exports from **Pencil**).

## Suggested layout

- `design/wait-app-design.pdf` — Full spec or screen PDFs
- `design/screens/` — Exported PNG/SVG per screen (splash, onboarding, home, venue-detail, etc.)
- `design/*.fig` — Pencil .fig source files (if provided)

See **`docs/Design-Intake.md`** for the intake process and how tech lead/subagents use these files.
